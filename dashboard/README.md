# 🛒 EcomIQ — E-Commerce Analytics Dashboard

Professional Streamlit dashboard for BSc Data Science project:
**E-Commerce Sales & Customer Analytics Using Machine Learning**

## Pages

| Page | Description |
|------|-------------|
| 🏠 Home | Platform overview — combined KPIs & charts |
| 🛍️ Meesho | Order status, state demand, product & SKU analytics |
| 📦 Amazon | Revenue trends, category, state & fulfilment analysis |
| 🤖 ML Models | Train Logistic Regression, Decision Tree, Random Forest |
| 👥 Customer | Survey — demographics, spending, satisfaction |
| 🔮 Prediction | Live order status prediction with confidence breakdown |

## How to Run

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Run the app
streamlit run Home.py
```

The app will open at http://localhost:8501

## Data Files Required (place in /data/)
- `meesho Orders Aug.csv`
- `meesho ForwardReports.csv`
- `Amazon Sale Report.csv`
- `E-Commerce Shopping Behavior Survey (Responses) - Form responses 1.csv`
